# patent-expiry-system
# ⚖️ 专利过期选品智能系统

基于USPTO（美国专利商标局）专利数据的选品工具，帮助亚马逊卖家发现即将过期或已过期的美国专利产品，寻找低成本选品机会。

---

## 🚀 功能特点

- 🔍 **USPTO官方数据源**：直接从美国专利商标局获取数据，无需API Key
- 💡 **机会分评估**：基于过期时间、品牌、IPC分类等维度自动评分
- 📊 **数据分析看板**：可视化展示申请人分布、机会分分布
- 📋 **选品库管理**：支持专利状态跟踪（待调研/样品中/已采购等）
- 📤 **Excel导出**：一键导出选品清单

---

## 📋 系统要求

- **操作系统**：Windows / macOS / Linux
- **Python版本**：3.8 或更高
- **网络要求**：需要VPN/代理访问USPTO（服务器在美国）
